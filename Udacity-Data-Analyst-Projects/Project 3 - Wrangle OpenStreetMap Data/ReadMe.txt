Data used in this exercise is the open streetmap file of Oxford, England, home to the Oxford University, and a place where I studied abroad past 
summer and really enjoyed. The link to the file is: 
https://mapzen.com/data/metro-extracts with searching of Oxford, England. 

There are 2 code files. The cleaning and auditing file presents the data entry and cleaning process while the query commands presents the query process. Answer to the questions are in the pdf file.
